Pythonのインストール方法

2020, 07, 20

Design Frontend Featured

Pythonのインストール方法について説明します。Pythonとはプログラミング言語の1つです。分かりやすい文法が特徴で、機械学習やWebアプリ開発などでも使われています。今回はWindows10にインストールします。Windows7, Windows8.1でも同様にインストールすることができます。

インストールファイルのダウンロード

まずは下記のリンクからPythonのダウンロードページを開きます。

・ Python-Download

このページはWindowsでアクセスしたらWindows用、MacでアクセスしたらMac用のダウンロードページが表示されます。

  • 最新版の32bit版をダウンロードする場合

    ページが表示されたら黄色いボタンになっている「Download Python 3.x.x」をクリックすることで最新版をダウンロードすることができます。

    • なお、上のボタンから最新版をダウンロードすると32bit版になります。64bitのpcで32bit版を使用しても特に問題はありませんが、64bit版のほうが大容量の処理ができます。なお、32bitのpcでは64bit版のPythonは起動しないので注意してください。

  • 最新版の64bit版や他のバージョンのPythonをダウンロードしたい場合

    ダウンロードページの中段にある「Looking for a specific release?」と書いてあるリストから、欲しいバージョンの名前かDownloadという場所をクリックします。今回は現時点（2020年7月）での最新版である「Python 3.8.4」の64bit版をダウンロードします。

    欲しいバージョンのページが表示されたら、下のほうにスクロールします。スクロールしていくと下のほうに「Files」というリストが現れます。今回は64bit版なのでこのリストの中から「Windows x86-64 executable installer」をクリックしてダウンロードします。32bit版は「Windows x86 executable installer」を、Macは「macOS 64-bit installer」をクリックしてください。ダウンロードしたインストールファイルはデスクトップやダウンロードフォルダ など、任意の場所に保存してください。

Pythonのインストール

任意の場所に保存したインストールファイルをダブルクリックで実行します。インストーラーが起動したら、一番下の「Add Python 3.8 to Path」というチェックボックスにチェックを入れてください。（入れなくても構いませんが、入れるとコマンドプロンプトやエディタなどから簡単にPythonを利用できるようになります。）

そのあと、「Install Now」をクリックすることでインストールが開始されます。インストールする場所など詳しく設定したい方は、その下の「Customize Installation」からインストールしてください。

インストール完了です。「Close」をクリックして閉じてください。
