フォントのサイズについて

2021, 04, 28

Design Frontend CSS Font

フォントのサイズについての備忘録です。「px」「%」「em」「rem」


フォントサイズの指定方法

CSSにおけるフォントサイズの指定方法は大まかに以下の二つ分けることができます。

  • 絶対値

    どんな場合でもその指定した値で表示される。

    例：「px」

  • 相対値

    親要素のサイズによって値が可変して表示される。

    例：「%」「em」「rem」

「px」

そもそも、pxとは画面に表示している色の最小単位のことです。画面全体のpx数はそれぞれデバイスによって違い、一般的なPCではフルHDの1920 × 1080pxがよく使われています。ちなみにHDは1280 × 720px、4Kは3840 ×2160pxです。

例えばフォントサイズを32pxと指定した場合、ほかの要素に左右されずに必ず32pxで表示されます。 


    html {
        font-size: 16px;  /* => 16px */
    }
    
    body {
        font-size: 32px;  /* => 32px */
    }
    
    h1 {
        font-size: 64px;  /* => 64px */
    }

「%」

%は親要素のフォントサイズを基準に、その何%なのかで大きさが決まります。

例えば親要素のフォントサイズが16pxの状態で、その子要素のフォントサイズに200%と指定した場合、子要素のフォントサイズは親要素16pxの200%の32pxの大きさで表示されます。 


    html {
        font-size: 16px;
    }
    
    body {
        font-size: 200%;  /* => 32px */
    }
    
    h1 {
        font-size: 200%;  /* => 64px */
    }

「em」

emも%と似ていますが、親要素のフォントサイズを１とした場合の、その相対サイズで大きさを指定します。

例えば親要素のフォントサイズが32pxの状態で、その子要素のフォントサイズに0.5emと指定した場合、子要素のフォントサイズは親要素16pxの0.5倍の8pxの大きさで表示されます。ちなみにCSSでは「0.5em」を「.5em」のように小数点前の0を省略することができます。 


    html {
        font-size: 16px;
    }
    
    body {
        font-size: 0.5em;  /* => 8px */
    }
    
    h1 {
        font-size: 2em;  /* => 16px */
    }

「rem」

remは「root em」の略です。つまりremとは、ルート要素のフォントサイズを1とした場合の相対サイズです。ちなみにルート要素とは基本的にhtmlセレクタを指します。

例えばルート要素のフォントサイズが16pxの状態で、対象要素のフォントサイズに2remと指定した場合、対象要素がどのような入れ子構造になっていても、フォントサイズはルート要素の2倍の32pxの大きさで表示されます。 


    html {
        font-size: 16px;
    }
    
    body {
        font-size: 2rem;  /* => 32px */
    }
    
    h1 {
        font-size: 2rem;  /* => 32px */
    }

ブラウザのデフォルトサイズ

ちなみにブラウザのデフォルトフォントサイズは16pxです。 


    html {
        font-size: 100%;  /* => 16px */
    }

まとめ

まとめると以下の通りです。

px サイズを絶対値指定
% サイズを親要素基準で相対値指定
em サイズを親要素基準で相対値指定
rem サイズをルート要素基準で相対値指定

一番使いやすいpxだけで表記をした場合、ブラウザによって文字の大きさが変わってしまうことがあります。また全体の文字の大きさを変えたい場合、一つ一つ変更していかなくてはならないので大変です。

それぞれ使いこなせるようにしたいですね。
